How it works

⌥ ⌘ Space Single press — briefly highlights the bubble to bring it to your attention.
⌥ ⌘ Space ×2 Double-tap — opens edit mode so you can type your current task.
Enter Save the text and return to click-through mode. Up to 20 characters.
Escape Cancel editing without saving.
